# 🚀 GeoStream - Real-Time Tracking Backend

GeoStream is a backend-focused real-time location tracking application built using Node.js, Express.js, Socket.IO, and Leaflet.js.

The project demonstrates real-time bidirectional communication using WebSockets, live geolocation tracking, dynamic map updates, and event-driven architecture.

---

# 🌟 Features

- Real-time location tracking
- WebSocket communication using Socket.IO
- Live location broadcasting
- Dynamic marker updates
- User disconnect handling
- Event-driven backend architecture
- Interactive map integration with Leaflet.js

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/raj11mm/Realtime-Geo-Tracker.git
```

---

## Move to Project Folder

```bash
cd GeoStream
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Server

```bash
nodemon app.js
```

or

```bash
node app.js
```
